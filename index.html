Movie Trailers

Watch movie trailers while viewing information about the movie. This website contains two pages. The Critics' Pick page allows the user to view the top 6 movie picked by NY Times critics, and subsequently view the information and a playlist of trailers associated with them. The Search page allows the user to do the same, but also input the title of the movie.
